Before You Install This Version
  1. Click Help in the menu bar of the main screen of your current version of PriDoc.
  2. Select the 'About PriDoc' option and note the Database version number. If your Database version is {{doc.prev_db_version}}, proceed with the download (see Instructions below).
  3. Otherwise, please contact support.

 

Instructions

Install the new version by carrying out  both the following steps ( A and B):

 

  A: On the PC or Server that hosts the PriDoc database 

  Download PriDoc Database Update (v2.33)

  1. Backup the PriDoc database using the backup facility in the PriDoc application (choose the Admin option in the menu bar of the main screen and then click the tab labelled ‘Backups'). It is essential that this first step is carried out successfully before you proceed to steps 2 and 3.
  2. Click the link directly above and then select 'Run' (or, for some browsers, 'Open').
  3. In the screen that appears, click the button labelled “Start Update”. The wizard will automatically action the required upgrade.

 

B: On all PCs (including the PC or Server that hosts the PriDoc database)

  Download PriDoc Application v33.2.0

  1. Click the link directly above and then select 'Run' (or, for some browsers, 'Open').
  2. In the menu that appears, choose the option called “Setup New Version of PriDoc Application” and the PriDoc application setup wizard should then begin. The wizard will automatically upgrade the PriDoc application to the new version.
