Security Levels 

Some of the information used by PriDoc will be confidential in nature (e.g. personnel records, pay details, etc.) and so it will be necessary to restrict some of the functionality to certain users. 

Rather than give every user yet another username to remember, access to the PriDoc modules is controlled through the use of just four passwords corresponding to four levels of access, as follows:

1. PriDoc Administrator - Unlimited access to all components of the PriDoc system.

2. Manager - Intermediate access to the PriDoc system.

3. Privileged User - More restricted access to the PriDoc system.

4. Standard User - Basic access to the PriDoc system.

A password for the ‘Standard User’ level is optional. If no password is specified, anyone may access PriDoc at the ‘Standard User’ level without a password
