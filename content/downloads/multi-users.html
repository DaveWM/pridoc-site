Utility for Creating PriDoc Users

PriDoc is designed for both single user and multi-user environments. In the latter, multiple users can connect across the network and make changes to the same PriDoc database. This means several users can work with the same database objects at the same time. 

A key issue when working with a database in a multi-user environment is access permissions. The permissions you, as a user, have for the database determine the extent of the work you can do with the database. For example, to make changes to objects in a database, you must have the appropriate write permissions for the database.

By default, as a matter of security, network users are not allowed access to the PriDoc database.  We have therefore produced a utility called "PriDoc_Access" that you are free to download below that enables you to grant access to the PriDoc database for specified Windows Users and Windows Groups.   

An audit trail of the creation/removal of each PriDoc user is written to the folder "Database Update Logs" in the ‘Emani Software' folder in the ‘Program Files' directory.

We hope you find this utility to be of value. As always, do please get in touch (via the contact methods given below) if you require any further information.

Instructions

 

  Download Utility for creating PriDoc Users

        

  1. Click the link above to start the download.
  2. Save the download to your computer by clicking the Save button.
  3. Copy the downloaded file to the machine that hosts the PriDoc database.
  4. Run as an administrator the downloaded executable "PriDoc_Access" and specify the name of the Windows User or Group who is to have access to the PriDoc database.
