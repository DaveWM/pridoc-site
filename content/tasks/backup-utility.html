Database Backup Utility

Backing up and maintaining your PriDoc database is extremely important and one of the best approaches for your organisation would be to create a scheduled task that carries out the backup and maintenance automatically at pre-allotted times. The task could be regularly scheduled (at least once per day) for specified times when the machine that hosts the database is known to be powered on. With this in mind, we have especially produced a backup/maintenance utility that you are free to download below. An audit trail of each backup is written to the folder "Database Backup Logs" in the ‘Emani Software' folder in the ‘Program Files' directory.

In this backup utility, we have also included the following maintenance jobs: history cleanup, rebuild of indexes and statistics update. This is so that these too can be actioned on a regular basis (i.e. automatically every time the backup runs). The audit trail for these maintenance jobs is written to the folder "Database Update Logs" in the ‘Emani Software' folder in the ‘Program Files' directory.

We hope you find this backup/maintenance utility to be of value. As always, do please get in touch (via the contact methods given below) if you require any further information.

Instructions

 

  Download Database Backup Utility

        

  1. Click the link above to start the download.
  2. Save the download to your computer by clicking the Save button.
  3. Copy the download to the machine that hosts the PriDoc database.
  4. Create a Scheduled Task to run as an administrator the downloaded executable "PriDoc_Backup" at regular times (e.g. daily).
  5. Consult Technical Support at Emani Software if further instructions are required.
