Utility for Creating Security Copies

Having backed up your PriDoc database (see Database Backup Utility), it is important that the PriDoc files and folders on the PriDoc Server (i.e. the machine on which the PriDoc database resides) are copied to a secure storage location. One of the best approaches for your organisation would be to create a scheduled task that creates the copies automatically at pre-allotted times. The task could be regularly scheduled (at least once per day) for specified times when the PriDoc Server is known to be powered on. With this in mind, we have especially produced a utility called "Security Copy" that you are free to download below that creates copies of the PriDoc files and folders in a user-specified target directory. For example, this target directory could be on a network drive that is regularly backed up to tapes which are then stored off-site or the target could be located on a separate PC (i.e. not the PriDoc Server). Using either of these locations for creating your security copies ensures that in the event of a failure affecting the PriDoc Server, your backup data is safely preserved elsewhere and can be used to restore your system to how it was when the latest security copy was created.

A configuration tool named “Setup Configuration File” is also supplied in the download so that the target directory for the copies can be easily specified.

An audit trail of each security copy is written to the folder "Database Backup Logs" in the ‘Emani Software' folder in the ‘Program Files' directory.

Instructions

 

  Download Utility for creating Security Copies

        

  1. Click the link above to start the download.
  2. Save the download to your computer by clicking the Save button.
  3. Open the download and unzip the files. 
  4. Copy all the unzipped files to the machine that hosts the PriDoc database.
  5. Run as an administrator the downloaded executable "Setup Configuration File" and specify the directory in which copies of the PriDoc data should be created.
  6. Create a Scheduled Task to run as an administrator the downloaded executable "Security Copy" at regular times (e.g. daily).
  7. Consult Technical Support at Emani Software if further instructions are required.
