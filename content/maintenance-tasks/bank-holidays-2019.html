This download sets up the Scheduler module in your PriDoc database with the expected Bank Holiday dates for calendar year 2019. There are eight permanent bank holidays in England and Wales in 2019 but note that, for holiday leave years that commence at the start of April, there are only 7 bank holidays in 2018-19 compared to 9 in 2017-18 due to the earlier Easter in 2018.

Running the maintenance task is a very straight forward and a very quick matter and involves carrying out the steps given below solely on the PC that hosts the PriDoc database. No action is needed on any other PCs.

  

Instructions

 

  Download Bank Holiday dates for 2019

  1. Click the link directly above to start the download. 
  2. Save the download to your computer by clicking the Save button.
  3. Run the maintenance script by carrying out the following steps: 

 

     On the PC that hosts the PriDoc database

  1. Backup the PriDoc database using the backup facility in the PriDoc application (choose the Admin option in the menu bar of the main screen and then click the tab labelled ‘Backups'). It is essential that this first step is carried out successfully before you proceed to step 2.
  2. Double click the file called 'Bank Holidays 2019' that you downloaded above and click 'Run'.
  3. Click the button labelled “Start Maintenance”. The wizard will automatically action the required task.
