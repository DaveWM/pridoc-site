Payroll module  – HMRC have created a new Transaction Engine service for processing Real Time Information (RTI) submissions and this service is now live. The PriDoc system has been updated to use the new live service and the software has been successfully tested against the new end points.

Payroll module  – HMRC encourages employers to access PAYE online regularly so they are aware of coding notices and important messages from HMRC. In conjunction with this advice and in order to assist you gain easy access to HMRC's outgoing notifications, a new tab titled 'HMRC Notices' has been added to the module's main screen for those customers subscribed to PriDoc's full payroll module. This new tab enables users to download quickly and easily the following types of notice directly from HMRC:

  • Tax Code notices (P6, P6B and P9)
  • Student Loan notices (SL1 and SL2)
  • Employer notices
  • RTI notices
  • Generic Notifications

For Tax Code and Student Loan notices, the PriDoc system will try to update the employee's record automatically with the new information, assuming a match on NI number and surname can be made. 

Backup Utility  – the General Data Protection Regulation (GDPR) comes into force in May 2018 and will regulate how personal data is stored, moved, handled and destroyed. To help address GDPR compliance, the compression and encryption of the database backups has been made stronger in this release. The main database backup file is now denoted by an EZZ file type.
